# Django blog

On this little blog the blogger can:
* sign in, sign out.
* add, edit and delete posts.
* approve comments to be displayed.

External users can:
* add comments (to be approved by the blogger).

----

This is a Python and Django exercise with AWS deployment.

Python `2.7.1`

Django `1.10`
